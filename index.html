Objetivos del Curso

• Desarrollar Sitios Web que satisfagan las necesidades del usuario.
• Analizar y operar lenguajes de Diseño web como HTML5 y CSS3.
• Operar de forma adecuada y administrar sistemas de control de versiones como GIT o GITHUB.
• Reconocer y utilizar diferentes metodologías de Diseño Web.

Temáticas de referencia

Los temas que trataremos como referencias principales en el siguiente Objeto Virtual de Aprendizaje son: HTML, CSS 3, GitHub.

Con el objetivo de ampliar la información, presentaremos dos videos de autoría propia sobre cómo crear nuestro primer Sitio Web y nuestro Primer Repositorio.

Información OVI

El OVI nos ofrece una iniciación básica en el campo del diseño web utilizando sistemas de control de versiones como GIT y la comunidad en línea GITHUB con su cliente para escritorio. Además de la utilización de los lenguajes de diseño web HTML5 y CSS3.

Objetivos del OVI

Buscamos con el presente Objeto Virtual de Aprendizaje que cualquier persona acceda a nuestro sitio y pueda aprender las generalidades de cómo hacer una página Web por medio de HTML y CSS, además explicaremos mediante diferentes lecturas y de forma interactiva la importancia de llevar el control de versiones de un proyecto a através de los repositorios que se pueden crear con la aplicación GitHub.
